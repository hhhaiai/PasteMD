# 快速开始

本指南将帮助你在几分钟内开始使用 PasteMD。

## 基本使用流程

使用 PasteMD 只需简单四步：

### 1️⃣ 启动应用

下载并安装 PasteMD 后，启动应用。程序会自动驻留在系统托盘。

**Windows**：在任务栏右下角看到 📋 图标
**macOS**：在菜单栏右上角看到 📋 图标

### 2️⃣ 复制 Markdown 内容

从任何地方复制 Markdown 格式的内容，例如：

- AI 对话网站（ChatGPT、DeepSeek、Kimi 等）
- Markdown 编辑器
- GitHub、博客文章
- 本地 `.md` 文件

::: tip 支持的内容类型
- 纯 Markdown 文本
- HTML 富文本（从网页复制）
- Markdown 表格
- `.md` 文件（从文件管理器复制）
:::

### 3️⃣ 打开目标应用

打开以下任一应用并将光标定位到你想插入内容的位置：

- Microsoft Word
- Microsoft Excel（如果是表格）
- WPS 文字
- WPS 表格（如果是表格）

### 4️⃣ 按下热键

按下热键（默认 `Ctrl+Shift+B`），PasteMD 会自动：

1. 读取剪贴板内容
2. 识别内容类型
3. 转换为合适的格式
4. 插入到光标位置

**完成！** 🎉

## 示例：从 ChatGPT 粘贴到 Word

让我们通过一个实际例子来演示：

### 场景

你在 ChatGPT 中询问数学问题，得到了包含公式的回答：

```markdown
# 二次方程求根公式

对于方程 $ax^2 + bx + c = 0$，其解为：

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

当 $\Delta = b^2 - 4ac$ 时：
- $\Delta > 0$：两个不同的实根
- $\Delta = 0$：一个重根
- $\Delta < 0$：两个共轭复根
```

### 步骤

1. **选择并复制**：在 ChatGPT 中选中回答内容，按 `Ctrl+C`（或 `Cmd+C`）
2. **打开 Word**：切换到 Word 窗口，点击你想插入内容的位置
3. **按热键**：按 `Ctrl+Shift+B`
4. **查看结果**：内容立即插入，公式完美显示

### 效果

- ✅ 标题格式正确
- ✅ 行内公式 $ax^2 + bx + c = 0$ 显示为数学公式
- ✅ 块级公式居中显示，格式完美
- ✅ 列表项保持原有结构

## 示例：粘贴表格到 Excel

### 场景

你复制了一个 Markdown 表格：

```markdown
| 姓名 | 年龄 | 职位 | 备注 |
|------|------|------|------|
| **张三** | 28 | *工程师* | `后端开发` |
| **李四** | 32 | *产品经理* | ~~已离职~~ |
| **王五** | 25 | *设计师* | `UI/UX` |
```

### 步骤

1. 复制表格内容
2. 打开 Excel 并选中 A1 单元格
3. 按 `Ctrl+Shift+B`

### 效果

- ✅ 表格结构完整
- ✅ **粗体**格式保留
- ✅ *斜体*格式保留
- ✅ ~~删除线~~格式保留
- ✅ `代码`格式保留（背景色+等宽字体）

## 配置热键

如果默认热键与其他应用冲突，可以自定义：

### 通过托盘菜单

1. 右键点击托盘图标
2. 选择"设置热键"
3. 在弹出的窗口中，按下你想使用的组合键
4. 点击"保存"

::: tip 热键建议
推荐使用包含 `Ctrl`（或 `Cmd`）+ `Shift` 的组合键，避免与常用快捷键冲突。

示例：
- `Ctrl+Shift+V`
- `Ctrl+Shift+M`
- `Ctrl+Alt+V`
:::

### 通过配置文件

你也可以直接编辑配置文件：

1. 右键托盘图标 → "编辑配置文件"
2. 修改 `hotkey` 字段：

```json
{
  "hotkey": "<ctrl>+<shift>+v"
}
```

3. 保存文件
4. 右键托盘图标 → "重载配置"

支持的修饰键：
- `<ctrl>` / `<cmd>`（macOS）
- `<shift>`
- `<alt>` / `<option>`（macOS）
- `<win>` / `<cmd>`（macOS）

## 托盘菜单功能

右键点击托盘图标，你可以：

### 基本功能

- **启用/禁用热键**：临时禁用热键监听
- **系统通知**：开启/关闭操作通知
- **保留文件**：转换后是否保留 DOCX/XLSX 文件

### 高级功能

- **Excel 功能开关**：启用/禁用表格自动粘贴到 Excel
- **设置热键**：打开热键设置窗口
- **打开保存目录**：查看转换后的文件
- **查看日志**：查看错误日志
- **编辑配置文件**：高级配置选项
- **重载配置**：应用配置文件的更改

### 其他选项

- **检查更新**：检查新版本
- **语言切换**：中文/英文
- **退出**：关闭应用

## 无应用兜底模式

如果你在没有打开 Word/Excel 时触发热键，PasteMD 可以：

### 自动打开（默认）

自动启动 Word 并插入内容。

### 仅保存

将转换后的文件保存到指定目录（默认 `~/Documents/pastemd`），不打开应用。

### 复制到剪贴板

将内容转换为富文本格式并复制到剪贴板，你可以在任何应用中粘贴。

### 无操作

仅显示通知，不执行任何操作。

### 配置方法

编辑配置文件：

```json
{
  "no_app_action": "open"  // 可选: "open", "save", "clipboard", "none"
}
```

## 常见问题

### 热键没有响应？

1. **检查热键是否启用**：托盘菜单 → "启用热键"（打勾表示已启用）
2. **检查热键冲突**：尝试更改为其他组合键
3. **重启应用**：退出后重新启动

### 公式显示不正确？

1. **检查 Pandoc 安装**：确保 Pandoc 已正确安装
2. **检查公式语法**：确保使用标准 LaTeX 语法
3. **查看日志**：托盘菜单 → "查看日志"查看错误信息

### Excel 粘贴没有反应？

1. **检查 Excel 功能是否启用**：托盘菜单 → "启用 Excel"
2. **确认是表格内容**：PasteMD 只识别标准 Markdown 表格
3. **尝试 Word**：表格也可以粘贴到 Word

### macOS 权限问题？

详见 [macOS 权限设置](/zh/macos/permissions)。
