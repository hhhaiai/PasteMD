# wps

::: warning 重要
由于 macOS 版 WPS 没有 AppleScript 自动化接口，本人暂时没有找到好的解决方案。

故目前 macOS 版 WPS 只能以 LaTeX 代码形式插入公式，如 `$E=mc^2$` 和 `$$公式$$`，需要手动转换为公式。

以下讲一下如何在 WPS 中手动转换为专业公式样式。

后续可能会提供一个wps宏来批量转换公式，方便大家使用。
:::

## 手动转换公式

1. 复制粘贴后，选中 LaTeX 代码形式的公式文本，包括 `$...$` 或 `$$...$$`
    比如:
    
    `$E = mc^2$`

    `$$\int_{-\infty}^{\infty} e^{-x^2} dx$$`
    
2. 在菜单栏选择"插入" → "公式"，你会看到公式被公式编辑器包裹
3. 点击被包裹的 LaTeX 代码并点击右边的小箭头下拉列表
4. 选择"转换为专业公式"
5. 你会看到 LaTeX 代码被转换为专业的公式显示

非常抱歉 macOS 版 WPS 目前无法实现自动转换公式的功能，感谢理解！
