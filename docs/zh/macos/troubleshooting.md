## 故障排查

::: warning 重要
在遇到问题前，请确保已正确设置所有权限，且应用已重启以应用权限更改。

本macos版仅在macos26下测试过，其他版本可能存在未知问题。

mac版本暂时支持不佳，还请谅解

有问题欢迎加QQ群：1073549447或者issue反馈
:::

### 热键不响应

1. **检查辅助功能权限**：系统设置 → 隐私与安全性 → 辅助功能
2. **检查热键是否启用**：菜单栏 → "启用热键"（应该有勾选）
3. **尝试更换热键**：避免与系统快捷键冲突
4. **重启应用**：退出后重新启动

### 无法检测应用

1. **检查录屏权限**：系统设置 → 隐私与安全性 → 屏幕录制
2. **刷新权限状态**：设置 → 权限 → 点击"刷新"
3. **查看日志**：菜单栏 → "查看日志"

### Word 插入 / Excel 写入失败

1. **检查自动化权限**：系统设置 → 隐私与安全性 → 自动化 → PasteMD
2. **重新授权**：重启 PasteMD，确保有权限
3. **确保 Office 应用已打开**：Word/Excel 必须在前台运行

### WPS 粘贴失败

1. **检查输入监控权限**：系统设置 → 隐私与安全性 → 输入监控
2. **重新授权**：重启 PasteMD，确保有权限
3. **确认 WPS 窗口在前台**：WPS 必须是活动窗口
